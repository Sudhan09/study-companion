4 · Find · Replace · Count

Locate, swap, and tally substrings inside a string.
Loop Week · Day 3 · Block A.3 · Subtopic 4

find vs index — pick by what you want on a miss

s = "hello world"

# FIND — returns -1 on miss (sentinel, no crash)
s.find("world")        →  6
s.find("o")            →  4         (returns the FIRST one)
s.find("zebra")        →  -1        (not found)

# INDEX — same when found, CRASHES on miss
s.index("world")       →  6
s.index("zebra")       →  ValueError: substring not found
Pick rule · .find() if you'll check the result yourself (if pos == -1: ...). .index() if you're certain it's there and want a crash if wrong.
Right-side versions · .rfind() and .rindex() search from the right end. Useful for finding the LAST occurrence — e.g. "a.b.c.txt".rfind(".") → 5 for the extension dot.

replace — swap every occurrence

s = "the cat sat on the mat"

s.replace("the", "a")             →  "a cat sat on a mat"
s.replace(" ", "_")               →  "the_cat_sat_on_the_mat"

# Limit how many swaps with a count argument
s.replace(" ", "_", 2)            →  "the_cat_sat on the mat"   (only first 2)

# Returns a NEW string — original unchanged
s                                  →  "the cat sat on the mat"
Chaining · multiple replaces in a row work fine: s.replace("a", "1").replace("b", "2").

count — non-overlapping occurrences

"hello".count("l")            →  2
"banana".count("a")           →  3

# The trap — count does NOT overlap
"ababab".count("ab")          →  3       (not 5)
"aaaa".count("aa")            →  2       (not 3)

# Walk-through for "ababab".count("ab"):
#   Found "ab" at pos 0 → count=1, jump past it to pos 2
#   Found "ab" at pos 2 → count=2, jump past it to pos 4
#   Found "ab" at pos 4 → count=3, jump past it to pos 6 (end)
#   Total: 3
Trap · count is non-overlapping. After each match, the search jumps past the matched substring. "aaaa".count("aa") is 2, not 3.

Real-world patterns

# 1. Check if substring exists (use `in` — simpler than find != -1)
if "error" in log_line:
    handle_error()

# 2. Find the file extension via rfind
filename = "report.final.pdf"
dot = filename.rfind(".")
ext = filename[dot+1:]                  →  "pdf"

# 3. Replace placeholders in a template
template = "Hello {name}, you owe {amount}"
template.replace("{name}", "Alice").replace("{amount}", "$50")
                                         →  "Hello Alice, you owe $50"

# 4. Count specific characters (vowels, digits, etc.)
text = "Hello World"
vowels = sum(text.lower().count(v) for v in "aeiou")    →  3

Lock these

  1. .find() returns -1 on miss. .index() crashes. Pick by what you want.
  2. .rfind() / .rindex() search from the right end — great for finding the last occurrence.
  3. .replace() swaps every occurrence by default. Pass a third argument to limit.
  4. .count() is non-overlapping. Jumps past each match.
  5. For "is this substring in here?" use in, not .find() != -1. Cleaner.
