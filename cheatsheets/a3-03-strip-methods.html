3 · Strip methods

Trim unwanted characters from the start and end. The middle is never touched.
Loop Week · Day 3 · Block A.3 · Subtopic 3

The three flavors

s = "   hello   "

s.strip()        →  "hello"          (both ends)
s.lstrip()       →  "hello   "       (LEFT end only — l = left)
s.rstrip()       →  "   hello"       (RIGHT end only — r = right)

# Default removes whitespace: spaces, tabs, newlines
"\n\t  hi  \n".strip()    →  "hi"

Stripping specific characters

# Pass characters to remove — "any of these" from the ends
"###hello###".strip("#")        →  "hello"
"...truth...".strip(".")        →  "truth"
"<>".strip("<>")         →  "title"        (any of < or >)

# The argument is a SET of characters, not a sequence to match whole
"abcXYZabc".strip("abc")        →  "XYZ"          (strips any of a/b/c)
"abcXYZcba".strip("abc")        →  "XYZ"          (same — order doesn't matter)</code></pre>
  <div class="tip"><b>Lock</b> · <code>.strip(chars)</code> reads <code>chars</code> as a set of "any of these," not as a substring to match. The order doesn't matter.</div>
</section>

<section class="card">
  <h2>The big trap — strip never touches the middle</h2>
<pre><code>"  hello  world  ".strip()      →  "hello  world"
                                    ↑ middle spaces stay!

"a#b#c#".strip("#")             →  "a#b#c"
                                    ↑ inner # stays, only trailing # goes

# To remove ALL of something — even in the middle — use .replace()
"  hello  world  ".replace(" ", "")    →  "helloworld"
"a#b#c#".replace("#", "")              →  "abc"</code></pre>
  <div class="trap"><b>Classic bug</b> · using <code>.strip()</code> to "remove all spaces." It only cleans edges. For interior, use <code>.replace(" ", "")</code>.</div>
</section>

<section class="card">
  <h2>Real-world patterns</h2>
<pre><code># 1. Clean user input
user_input = "  alice@example.com  \n"
clean = user_input.strip()                      →  "alice@example.com"

# 2. Read lines from a file (each line ends with \n)
for line in open("data.txt"):
    line = line.strip()                         # remove the trailing newline

# 3. Remove quotes around a string
'"hello"'.strip('"')                            →  "hello"

# 4. Right-strip only — useful when leading whitespace is intentional
"    indented code\n".rstrip()                  →  "    indented code"</code></pre>
</section>

<section class="summary">
  <h2>Lock these</h2>
  <ol>
    <li>Default <code>.strip()</code> removes whitespace from both ends.</li>
    <li><code>l</code>strip = left only, <code>r</code>strip = right only.</li>
    <li><code>.strip(chars)</code> reads chars as a <b>set</b> — any of those, in any order, from the ends.</li>
    <li><b>The big trap:</b> middle characters are never touched. For interior, use <code>.replace()</code>.</li>
    <li>Most common use: cleaning the trailing <code>\n</code> off file lines, or trimming user input.</li>
  </ol>
</section>
